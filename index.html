Sriprabha Ramanarayanan

Senior Project Engineer, HTIC - IIT Madras

I'm a Senior Project Engineer in the Image Computing Group of the Healthcare Technology Innovation Centre, Indian Institute of Technology (IIT) Research Park, since 2013. I'm also a final year (10th semester) full-time Ph.D. scholar at the IIT Madras working with Prof. Mohanasankar Sivaprakasam. I'm one of the technical leads for the Medical Imaging projects using deep learning for MRI. My work focuses on mathematical method developments for real-world solutions in medical imaging applications using conventional, machine learning, and deep learning-based techniques. I'm also guiding around 10 research scholars of the institute of which 5 are women. Previously, I worked in the industry for around eight years (2005 - 2013) in general and medical imaging in 1. HCL Technologies, Chennai, as a Senior Technical Lead in Imaging Center of Excellence, 2. Philips Healthcare, Philips Innovation Campus, Bangalore as a Technical Specialist on Preclincal Imaging and Radiation Oncology Systems, and 3. Vortex Engineering on Biometric Fingerprint Image Analysis, IITM Research Park as a Design Engineer. In my initial years of career, I worked as a Project Associate in the Industrial Consultancy and Sponsored Research at IITM, parallely did full-time my MS (by research) at the IIT, Madras.

Meta-learning for MRI Reconstruction

Meta-learning for MRI Reconstruction (January 2020 to December 2022)

Conventional deep learning models are task-specific and need retraining for varying acquisition settings. This project explores meta-learning as an effectively tool to develop adaptive learning models for MRI reconstruction. Furthermore, we analyze the generalization ability of model-based meta-learning methods based on weight-learning approaches in deep learning to unseen images acquired with deviated scan settings.
Nature of work: Data collection and curation, training and validation of neural network architectures and image visualization
Programming environment – Pytorch

Biometric Fingerprint Image Analysis

Design of biometric fingerprint authentication system for ATMs (MS project work - June 2006 to July 2007)

The focus of this research is to design a Fingerprint Authentication System and to optimize its performance in terms of speed and False Rejection Rates (FRR) without degrading its False acceptance Rates.
The work involves 1) The development of an optimal feature extraction scheme based on SVD. 2) Improving the speed of user fingeprint registration process by reducing the redundancies in computation of fingerprint features.
3) Image acquisition system development to capture fingerprints, Interface programming for keypad, LCD display, Image pre-processing and forwarding over Ethernet at the DSP side.
4) Server side C programming to receive images and user registration / authentication completion. Integration of fingerprint encryption/decryption and H.263 encoding/decoding.

Nature of work: C and assembly language programming for Blacfkin DSP. Algorithm development and testing in Matlab, OpenCV and C

Publications.

DC-WCNN: A deep cascade of wavelet based convolutional neural networks for MR Image Reconstruction (ISBI 2020)

Sriprabha Ramanarayanan, Balamurali Murugesan, Keerthi Ram, Mohanasankar Sivaprakasam

An encoder-decoder based CNN that uses Wavelet Packet Transform iin place of pooling layers to address the problem of information loss due to sub-sampling.

MAC-ReconNet: A Multiple Acquisition Context based Convolutional Neural Network for MR Image Reconstruction using Dynamic Weight Prediction (MIDL 2020)

Sriprabha Ramanarayanan, Balamurali Murugesan, Keerthi Ram, Mohanasankar Sivaprakasam

MIDL 2020, long oral, Runner-up Best paper award, one of the 4 awarded papers chosen from 18 long oral papers out of 146 submissions (that is, 12.2% acceptance rate for oral papers)

MAC-ReconNet is an adaptive MRI Reconstruction neural network architecture that uses dynamic weight prediction (DWP) network that completely decouples the reconstruction CNN from learning. The DWP module takes acquisition context information as input and learns the context-specific weights of the reconstruction module which changes dynamically with context at run time.

Paper link         Code link

MCI-HyperNet: A multiple contextual information-based adaptive weight learning network for controllable image reconstruction (Neurocomputing 2023)

Sriprabha Ramanarayanan, Balamurali Murugesan, Arun Palla, Keerthi Ram, Ramesh Venkatesan, Mohanasankar Sivaprakasam

A coupled network with meta-learning capabilities of scalability, adaptability and extensibility. The network consists of a backbone reconstruction network and hypernetwork to infuse contextual knowledge of acquisition settings into the backbone network.

Paper link         Code link

HyperCoil-Recon: A Hypernetwork-based Adaptive Coil Configuration Task Switching Network for MRI Reconstruction (ICCVW CVAMD 2023)

Sriprabha Ramanarayanan, Mohammad Al Fahim, Rahul G. S., Amrit Kumar Jethi, Keerthi Ram, and Mohanasankar Sivaprakasam

Parallel MRI involves dynamic coil configurations i.e. varying sensitivities and number of the coils. HyperCoil-Recon encodes varying coil configurations in a multi-tasking perspective, posing each configuration as a task.

KM-MAML: Generalizing Supervised Deep Learning MRI Reconstruction to Multiple and Unseen Contrasts using Meta-Learning Hypernetworks (Applied Soft Computing, Special Issue on Evolutionary Deep Learning 2023, Code Reproducibility badge)

KM-MAML is a hybrid learning model that uses optimization-based and model-based meta-learning to adaptive and multimodal MRI reconstruction.

SFT-KD-Recon: Learning a Student-friendly Teacher for Knowledge Distillation in Magnetic Resonance Image Reconstruction (MIDL 2023, Code Reproducibility score > 5)
